# Service Hub

A modern **Multi-Vendor Freelance Marketplace** built with **Laravel** and **React.js**, inspired by platforms like Fiverr and Upwork. The platform allows clients to discover services, place orders, and manage projects, while freelancers can create gigs, showcase their portfolios, and track their earnings through a dedicated seller dashboard.

---

## Features

### 👤 User

- Secure User Registration & Login
- Search Services
- View Service Details
- Place Orders
- Leave Reviews

### 💼 Seller

- Become a Seller
- Create, Edit & Delete Gigs
- Portfolio Management
- Earnings Dashboard
- Order Management

### 🛠️ Admin

- Admin Dashboard
- Manage Users
- Manage Services/Gigs
- Monitor Orders

## 🚀 Installation

### Clone the Repository

```bash
git clone <repository-url>
cd Service-Hub
```

### Backend Setup

```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
